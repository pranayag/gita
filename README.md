## Bhagvad Gita App
You can visit the app [here](https://rossoskull.me/gita)


This is a ReactJS app, which uses the [bhagavadgita.io API](https://bhagavadgita.io/api/).

### Running on your local system
#### Installing & setting up
``` sh
$ npm install
```
#### Running the app
``` sh
$ npm start
```
#### Building the app
```sh
$ npm run build
```
